# Changelog

## 1.0.0 (2026-04-17)

- Initial release
- Native GUI window with embedded webview for Windows (WebView2), Linux (WebKitGTK), and macOS (WKWebView)
- Window title synced from HTML `<title>` element
- `window.close()` support from JavaScript
- `onClose` callback
- `node-gui-pack` CLI for bundling apps into standalone executables
