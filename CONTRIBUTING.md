# Contributing to node-gui

Thanks for your interest in contributing!

## Getting started

```bash
git clone https://github.com/tomika/node-gui.git
cd node-gui
npm install
npm run build
npm test
```

## Development workflow

1. Fork the repository and create a feature branch.
2. Make your changes with clear, focused commits.
3. Ensure `npm test` passes.
4. Open a pull request against `main`.

## Reporting bugs

Open an issue at https://github.com/tomika/node-gui/issues with:
- Your OS and version
- Node.js version (`node -v`)
- Steps to reproduce
- Expected vs actual behavior

## Code style

- Use `'use strict'` in all JS files.
- Follow existing formatting (2-space indentation, single quotes).
- No linter is enforced yet — just keep it consistent.
