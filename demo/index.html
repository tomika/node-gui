Collapsible sections — window auto-resizes to fit

Toggle sections; the native window height follows the content.

About this demo small

This window uses onSizeChanged with axes: 'height' and scrollbarGutter: 'stable' to grow and shrink only its height as you expand and collapse panels.

Features medium
  • Native window on Windows, macOS, and Linux
  • Embedded WebView2 / WKWebView / WebKitGTK
  • Auto height fit driven by the page content
  • Width is preserved across content changes
